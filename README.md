# Mansions of Madness: The Cursed Bone Cabinet

A modular, high-density storage system for *Mansions of Madness 2nd Edition* map tiles. Designed to look like an eldritch artifact fashioned from carved bone and cold iron.

## 💀 Thematic Vision
This organizer is designed as a **"Cursed Cabinet"**. 
- **Pillars**: Simulated "Cold Iron" (Matte Black with silver dry-brushing).
- **Trays**: "Aged Bone" (Cream/Beige filament with a dark wash).
- **Labels**: "Eroded Typewriter" style using the *Special Elite* font.

## 🛠 Project Structure
- `mom_assembly.scad`: Master preview of the full dual-tower system.
- `mom_pillar_left.scad` & `mom_pillar_right.scad`: Dedicated files for STL export.
- `mom_rail_lib.scad`: The core mechanical engine for the mirrored pillars.
- `mom_config.scad`: Customizable hardware specs (tray counts, clearances, etc.).

## 📏 Specs
- **Capacity**: 46 total trays (Two towers of 23).
- **Tray Size**: 270mm x 137mm.
- **Clearance**: 1mm total wiggle room for smooth operation.
- **Plywood Shell**: Designed for 5mm (top/bottom) and 3mm (back) plates.

## 🚀 Printing Instructions
1. **Pillars**: Print standing vertically for maximum strength. Use a brim for stability.
2. **Trays**: Print with the bottom on the bed. 
3. **Labels**: Use the *Special Elite* font. For best results, use a paint pen to highlight the raised eroded text.
